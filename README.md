# Over Basquete — XScores real

Versão nova com:

- lista de jogos reais do dia
- clique no jogo para abrir a tela H2H
- últimos jogos das duas equipes
- confrontos diretos
- tabela de cálculo estilo planilha com dados reais puxados do XScores

## Como rodar

```bash
npm install
npm start
```

Depois abra:

```bash
http://localhost:3000
```

## Deploy no Render

- Build Command: `npm install`
- Start Command: `npm start`

## Estrutura importante

A pasta precisa ser exatamente `public`.
